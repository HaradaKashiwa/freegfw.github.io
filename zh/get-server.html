选购要求与避坑指南

在您购买和创建云服务器时，请务必确认满足以下核心条件：

  • 操作系统：任意最新版本皆可，强烈推荐选装 Ubuntu 或 Debian 原版纯净系统。
  • 网络要求：服务商必须为您分配独立的「公网 IPv4 地址」。
🤝

FreeGFW 是一个完全开源的项目，自身不产生任何收益，与任何云服务商均无利益绑定。如果您希望支持我们的项目，可以使用带有返利的链接购买服务器，这样服务商会给予我们少量佣金以维持项目运转。如果您介意此类形式，完全可以使用无返利的链接直接购买。
